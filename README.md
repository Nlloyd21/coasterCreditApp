# Project Title (Update)

This is an app where the user can keep track of their coaster credit count after the ride a new ride.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download and install XCode on MacOS
2.Clone the repository project. 
3.Download the simulator
4. Press the run button.
5. THe simulator will pop up. Open the app on the simulator iPhone.

Instructions for using the software:

1. Press the + button in the corner.
2. Type in the name of the coaster, park name, and area of the park. 
3. Click save
4. Will then show up on the main home page.
